# ChatGPT Review Sentiment Analysis

This project analyzes user reviews of AI/ChatGPT apps and classifies them as:
✅ Positive  
❌ Negative  

It uses advanced NLP + ML to understand review sentiment from real-world text (mixed languages supported ✅).

---

## Tech Stack

| Component | Library |
|----------|----------|
| Language | Python |
| Text Preprocessing | NLTK |
| Embeddings | SentenceTransformer (MiniLM) |
| ML Model | Logistic Regression |
| Visualization | Matplotlib, Seaborn |
| Translation | GoogleTranslator |

---

## Key Features

✔ Cleans reviews & removes noise  
✔ Translates non-English → English  
✔ Sentence embeddings → better meaning capture  
✔ Class weight balancing → avoids biased learning  
✔ Saves model for deployment
